# Fix React Error #310 (Infinite Re-render) in Capacitor App

## Root Cause
1. `useRef(false)` guard in AuthContext prevents re-subscription after StrictMode unmount/remount, leaving `loading=true` forever
2. Context value object recreated every render (unmemoized functions) causes cascading re-renders

## Fix
**File:** `src/contexts/AuthContext.tsx`

1. Remove `useRef` guard — let `onAuthStateChanged` re-subscribe normally (it's idempotent)
2. Wrap `signInWithGoogle`, `signInWithEmail`, `signUpWithEmail`, `signOut` in `useCallback`
3. Memoize context value with `useMemo`

## Verification
- Build: `vite build` + `cap sync android` + `gradlew assembleDebug`
- Test: Install APK, open app — should show login screen without crash
